# Web Reconnaissance Task

## Overview
This repository documents a web reconnaissance task completed as part of cybersecurity training. All testing was performed on authorized practice targets only.

## Tools Used
- **dnsx** — Active subdomain brute forcing
- **massdns** — Fast DNS resolution
- **ffuf** — Directory, file and parameter fuzzing
- **wafw00f** — WAF detection
- **katana** — JavaScript crawling
- **gau** — Historical JS file discovery
- **httpx** — Live JS file verification
- **theHarvester** — Email and subdomain harvesting
- **WhatWeb** — Technology fingerprinting

## Targets Used
- testphp.vulnweb.com (Acunetix authorized test site)
- vulnweb.com (Acunetix authorized test site)
- nmap.org (public site, passive recon only)
- bbc.com (public site, passive recon only)

## What Was Covered
- Passive subdomain discovery
- Historical URL enumeration
- Active subdomain brute forcing
- Web directory and parameter fuzzing
- WAF detection
- JavaScript file analysis
- API endpoint extraction
